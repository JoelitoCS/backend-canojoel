## Estructura del Proyecto

El proyecto está organizado en tres ejercicios principales:

### Ejercicio 1: Filtros y Visualización

Aplicación React básica con filtrado de alumnos por promoción y nombre.

**Tecnologías:**

- React + Vite
- Tailwind CSS
- shadcn/ui

### Ejercicio 2: CRUD y Autenticación

Ampliación con operaciones CRUD completas y sistema de autenticación. Implementa control de acceso donde usuarios no logueados pueden ver alumnos, pero solo administradores pueden gestionarlos.

**Tecnologías:**

- React + Vite
- localStorage para persistencia
- shadcn/ui para formularios y modales

### Ejercicio 3: API Node.js + MongoDB

Backend completo con API REST y conexión a MongoDB Atlas.

**Tecnologías:**

- Node.js + Express
- MongoDB Atlas
- Mongoose

### Tutorial REST Client

Guía completa para probar la API directamente desde Cursor usando REST Client.

**Tecnologías:**

- REST Client (extensión de Cursor/VS Code)
- Archivos `.http` para peticiones HTTP

### Tutorial Fetch API

Guía completa para aprender a usar la Fetch API en JavaScript, incluyendo peticiones GET, POST, PUT, DELETE, manejo de errores e integración con React.

**Tecnologías:**

- JavaScript (ES6+)
- Fetch API
- React (ejemplos de integración)
